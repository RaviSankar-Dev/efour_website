# E-Four Website

A modern, high-performance web application for ticket management and booking, featuring a premium 3D user interface.

## 🚀 Features

- **Premium UI/UX**: Built with React, Tailwind CSS, and Framer Motion for smooth animations.
- **3D Graphics**: Integrated Three.js and React Three Fiber for immersive visual elements.
- **Mobile Support**: Fully responsive design with Capacitor for Android integration.
- **PWA Ready**: Progressive Web App support for offline capabilities.
- **Admin Dashboard**: Comprehensive management system for orders and tickets.
- **Real-time Scans**: QR code scanning supported via `html5-qrcode`.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) (Vite)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **3D Engine**: [Three.js](https://threejs.org/) / [React Three Fiber](https://github.com/pmndrs/react-three-fiber)
- **Mobile Platform**: [Capacitor](https://capacitorjs.com/)
- **Database/Backend**: [Supabase](https://supabase.com/) & [Express](https://expressjs.com/)

## 🏁 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/RaviSankar-Dev/efour_website.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Setup environment variables:
   Create a `.env` file in the root and add your configuration.

4. Start the development server:
   ```bash
   npm run dev
   ```

## 📦 Build & Deploy

- **Web Build**: `npm run build`
- **Mobile (Capacitor)**: `npx cap sync`
